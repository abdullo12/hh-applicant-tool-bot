## 🔑 ШАГ 5: Получить токены HH (5 минут)

### Вариант А: Через Termux (Android)

**1. Установите Termux:**
```
Откройте в браузере:
https://f-droid.org/packages/com.termux/

Нажмите "Download APK"
Установите
```

**2. Откройте Termux и выполните:**
```bash
pkg update
pkg install python git
git clone https://github.com/s3rgeym/hh-applicant-tool
cd hh-applicant-tool
pip install -e .
hh-applicant-tool auth
```

**3. Введите данные:**
```
👤 Введите email или телефон: ваш@email.ru
📨 Код был отправлен
📩 Введите код: 1234
🔓 Авторизация успешна!
```

**4. Скопируйте токены:**
```bash
cat ~/.config/hh-applicant-tool/config.json
```

**Результат:**
```json
{
  "token": {
    "access_token": "USER...",
    "refresh_token": "USER...",
    "access_expires_at": 1234567890
  }
}
```

📋 **СКОПИРУЙТЕ ВСЁ СОДЕРЖИМОЕ!**

### Вариант Б: Попросите друга с ПК

Пусть выполнит:
```bash
pip install hh-applicant-tool
hh-applicant-tool auth
```

И отправит вам файл `config.json`

## 🆘 Проблемы?

**Бот не отвечает:**
1. Проверьте Railway → Logs
2. Убедитесь что добавили все переменные
3. Проверьте что config.json создан

**Ошибка при /apply:**
1. Проверьте cover_letter.txt
2. Убедитесь что авторизованы на HH
3. Возможно достигнут лимит (200/день)

**Нужна помощь:**
- [Telegram канал](https://t.me/hh_applicant_tool)
